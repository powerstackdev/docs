## What is Power Stack?

Power Stack is an open-source digital experience platform (DXP) project.

Built on open data and modern web technologies, Power Stack revolutionizes tools for marketing and development teams giving flexibility to put customers at the heart of their digital portfolio.

## Try the pre-alpha today!

Hopefully, if you've read this far, you're convinced and want to try Power Stack.  We've tried to make development and setup as easy as possible using Docksal.

Our 1-command install generates a preconfigured ready-to-use project with all the needed dependencies.
